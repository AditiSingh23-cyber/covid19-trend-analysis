# COVID-19 Trend Analysis

## Project Overview
This project analyzes global COVID-19 trends using Python data analysis tools.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Steps Performed
1. Data cleaning and preprocessing
2. Exploratory Data Analysis (EDA)
3. Visualization of trends
4. Country-wise comparison

## Dataset
Public COVID-19 dataset used for analyzing global case trends.
